
## 🚀 Cara Mempublikasikan
 
### Opsi 1: Hosting Tradisional (cPanel/FTP)
 
1. Download ketiga file (index.html, style.css, script.js)
2. Upload ke folder public_html di hosting Anda
3. Akses melalui domain Anda
4. Done! Website sudah live
 
### Opsi 2: GitHub Pages (GRATIS)
 
1. Buat repository baru di GitHub
2. Upload ketiga file ke repository
3. Buka Settings > Pages
4. Pilih branch 'main' sebagai source
5. Save - GitHub akan memberikan URL gratis
6. Website live di: `https://username.github.io/repository-name/`
 
### Opsi 3: Netlify (GRATIS + Custom Domain)
 
1. Buka [netlify.com](https://netlify.com)
2. Drag & drop folder berisi ketiga file
3. Atau connect dengan GitHub repository
4. Netlify otomatis deploy
5. Domain gratis: `https://site-name.netlify.app`
6. Bisa tambah custom domain sendiri
 
### Opsi 4: Vercel (GRATIS)
 
1. Buka [vercel.com](https://vercel.com)
2. Import project dari GitHub
3. Atau upload file langsung
4. Auto-deploy setiap update
5. Domain gratis: `https://site-name.vercel.app`
 
## 🛠️ Cara Customization
 
### Mengubah Teks
 
Buka `index.html` dan edit bagian yang diinginkan:
 
```html
<!-- Hero Title -->
<h1 class="hero-title font-display">Crafting</h1>
<h1 class="hero-title-accent font-display">Distinction</h1>
 
<!-- Description -->
<p class="hero-description font-body">
    We create refined digital experiences...
</p>
```
 
### Mengubah Warna
 
Buka `style.css` dan ubah warna di variabel:
 
```css
/* Cari dan ganti warna-warna ini */
background-color: #0a0a0a;  /* Background utama */
color: #e8e5df;              /* Text color */
color: #d4af37;              /* Gold accent */
```
 
### Mengubah Foto Background
 
Buka `style.css` dan ganti URL gambar:
 
```css
/* Hero Section */
.hero-bg-image {
    background-image: url('GANTI_DENGAN_URL_FOTO_ANDA');
}
 
/* CTA Section */
.cta-bg-image {
    background-image: url('GANTI_DENGAN_URL_FOTO_ANDA');
}
```
 
### Menambah Section Baru
 
Copy struktur section yang ada di `index.html`:
 
```html
<section class="custom-section">
    <div class="container">
        <div class="section-header">
            <p class="section-label font-body">LABEL</p>
            <h2 class="section-title font-display">Title</h2>
        </div>
        <!-- Konten section Anda -->
    </div>
</section>
```
 
### Mengubah Font
 
Di `index.html`, ganti Google Fonts link:
 
```html
<link href="https://fonts.googleapis.com/css2?family=FONT_BARU&display=swap" rel="stylesheet">
```
 
Lalu di `style.css`, update:
 
```css
.font-display {
    font-family: 'FONT_BARU', serif;
}
```
